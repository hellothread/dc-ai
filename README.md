# DC-AI-Bot

## 项目描述
基于DeepSeek AI的智能信息获取和回复机器人，能够自动获取用户信息并生成个性化内容。

## 技术栈
- Python
- DeepSeek AI API

## 功能特点
- AI生成：利用DeepSeek AI模型生成定制化回复

## 安装步骤
1. 克隆此仓库：
   ```bash
   git clone https://github.com/hellothread/dc-ai.git
   ```
2. 进入项目目录：
   ```bash
   cd dc-ai
   ```
3. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```
